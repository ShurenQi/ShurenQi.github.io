Biography

Shuren Qi received the B.A. and M.Sc. degree from Liaoning Normal University (LNNU), Dalian, China, in 2017 and 2020 respectively. He is currently pursuing the Ph.D. degree in cyberspace security at Nanjing University of Aeronautics and Astronautics (NUAA), Nanjing, China. His research interest includes the image analysis and representation using polynomial expansions, and the advanced multimedia forensics and security techniques.

Research Interests

  • Pattern Recognition { Image Representation { Moments and Moment Invariants } }
  • Information Forensics and Security { Image Authentication { Watermarking and Forgery Detection } }
